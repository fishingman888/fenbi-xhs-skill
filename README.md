# FenBi XHS Skill

一个面向小红书内容创作的 Codex Skill，按“审核 → 诊断 → 保真优化 → 发布前自检”的固定流程处理图文笔记、文章、口播稿、视频脚本、教程和产品测评。

## 核心能力

- 内容识别：判断内容形式、主题、目标用户、核心价值和商业属性
- 事实锚定：锁定人名、品牌、数据、经历、合作关系等不可擅改信息
- 平台底线审核：识别虚假夸大、诱导互动、站外导流、标题错配及高风险领域表达
- 四维诊断：检查创作意图、视听与呈现、主旨清晰度和用户获得感
- 保真优化：在不虚构事实、不篡改数据、不改变原意的前提下完成整稿改写
- 发布前自检：重新检查标题、事实、风险表达、商业关系和内容完整度

## 规则分层

本 Skill 严格区分三类内容：

1. 平台底线：发布前必须优先排查的风险。
2. 优质内容标准：用于判断真实性、清晰度、呈现和获得感。
3. 非官方增长建议：标题、节奏、封面等运营经验，不冒充平台规则或算法要求。

## 使用示例

```text
使用 $fenbi-xhs-skill 审核、诊断并保真优化下面这篇小红书笔记：

[粘贴你的标题和正文]
```

Skill 会固定输出：

1. 内容识别
2. 事实锚定
3. 平台底线审核
4. 官方优质内容标准诊断
5. 优化策略
6. 完整保真优化稿
7. 改动说明
8. 发布前自检

## 安装

将仓库中的 `fenbi-xhs-skill` 文件夹复制到 Codex Skills 目录，然后重新启动或刷新 Codex。

```text
~/.codex/skills/fenbi-xhs-skill/
```

## 重要边界

- 不虚构事实、经历、数据、收入、销量、粉丝或案例
- 不承诺流量、推荐、爆款或平台审核结果
- 不冒充小红书官方审核系统
- 不把课程创作建议写成平台硬性违规条款
- 纯文字输入时，不假装判断真实画面和音频质量
- 医疗、金融等高风险内容需结合发布时的最新规则和专业意见

## 说明

“官方优质内容标准”部分依据公开平台资料及用户提供的小红书创作课程内容整理，用于内容诊断和编辑，不代表小红书官方授权、认证或完整审核规则。平台规则可能更新，发布前请以小红书最新公开规则为准。
